Setup - Arduino
---------------
1. Install Arduino IDE 1.8.1
   https://www.arduino.cc/en/Main/Software

2. Copy ./Arduino/Micromouse-Arduino/libraries folder to Arduino user libraries

2. Install Arduino IDE for Visual Studio
   https://marketplace.visualstudio.com/items?itemName=VisualMicro.ArduinoIDEforVisualStudio

3. Launch Micromouse-Arduino.sln

4. Configure Visual Micro (vMicro) to use "Arduino 1.6/1.8" and "Arduino Leonardo".


Setup - Unity
-------------
1. Install Unity v5.5.x

2. Launch Unity and point the project folder to ./Unity
